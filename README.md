# forjapu

ForJapu is a personalized reminder and care web app made for two close users to stay connected in a cute, simple, and supportive way. It creates a shared space where one user can send reminders, encouragement, and thoughtful updates, while the other user can view them, respond, and stay on track with daily routines.

The app includes role-based experiences for both users, making it feel more personal and interactive. One side can create and manage reminders, monitor progress, and send updates through a live whiteboard. The other side can receive reminders, mark them as completed, and view their daily schedule in a soft and friendly interface. This makes the app feel less like a strict productivity tool and more like a warm companion for daily life.

A special feature of ForJapu is the shared whiteboard, where users can draw, add stickers, write text, and send cute visual messages in real time. This adds a fun and emotional touch beyond normal reminder apps. The app also includes profile personalization, notification settings, reminder tracking, and dashboard views that help monitor completion progress over time.

The overall design focuses on a cozy, playful, and affectionate experience, with soft colors, simple navigation, and mobile-friendly layouts. It is built to work smoothly across different screen sizes and can be hosted online for easy access by both users.

ForJapu is more than just a reminder app. It is a small shared digital space made for care, encouragement, fun interaction, and staying connected through everyday moments.
